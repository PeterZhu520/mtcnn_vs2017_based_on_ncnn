
## 前言
网上大多的ncnn的vs版本代码都比较老，同时为了在Windows下单步调试学习mtcnn模型，因此把网上的资源整合了一下，分享出来。

## 使用


直接git clone 到本地，然后使用vs2017打开ncnn文件夹下的ncnn.sln，编译运行即。（依赖的Opencv已经添加）

注：目前只测试了test_picture函数，test_video并未测试
